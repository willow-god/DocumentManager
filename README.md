# 武汉理工大学专属档案管理系统

#### 介绍
JAVA程序设计实验，其中借鉴了CSDN叶小岚的文章，仅作私用

#### 安装教程

1.  clone
2.  在电脑里创建好相应的文件夹（不创好像也行，就是第一次运行报错）
3.  运行

#### 我的更新

2. 进行了错误地抛出，但是还是没有进行具体内容，使用哈希表存储内容
